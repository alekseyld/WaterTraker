# WaterTraker

Предназначено для устройств с ОС Android. 
Написано на языке программирования Kotlin с использованием архитектурного стиля MVP.
Сторонние библиотеки:
- Retrofit 2
- Dagger 2
- RxJava 2
- Room
- AnyChart
